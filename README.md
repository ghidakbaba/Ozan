# Ozan Akbaba Website

A modern, responsive, GitHub Pages-ready fan website about Turkish actor and composer **Ozan Akbaba**.

## Files

- `index.html` — page structure and content
- `style.css` — full responsive design
- `script.js` — animations, mobile menu, filmography filtering

## How to upload to GitHub Pages

1. Create a new GitHub repository, for example `ozan-akbaba-website`.
2. Upload `index.html`, `style.css`, and `script.js` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Save. GitHub will give you a live website link.

## Important note about images

This project intentionally does not include copyrighted actor photos. The hero uses a stylized initials card. If you add images, only use photos you own, have licensed, or have permission to publish.

## Customization

Change colors in `style.css` inside the `:root` section. Add more filmography items in `script.js` inside the `works` array.
